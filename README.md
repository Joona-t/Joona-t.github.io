# LoveSpark Portfolio Site

Static single-page portfolio that matches the LoveSpark Retro Pink Chrome theme.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `og.svg` (1200x630 source)

## Edit placeholders
Open `index.html` and replace:
- `YOURUSERNAME`
- `YOURHANDLE`
- `you@example.com`
- `PASTE_CHROME_EXTENSION_LINK`
- `PASTE_CHROME_THEME_LINK`
- `PASTE_FIREFOX_LINK`
- `PASTE_EDGE_LINK`
- `PASTE_REPO_LINKS`

## Social image
1. Export `og.svg` to `og.png` at `1200x630`.
2. Put `og.png` in the site root.
3. Keep the `og:image` URL pointing to `https://YOURUSERNAME.github.io/og.png`.

## Local preview
Double-click `index.html` or open it in your browser.

## Deploy to GitHub Pages
1. Push this site to a repository named `YOURUSERNAME.github.io`.
2. In GitHub, open: `Settings` -> `Pages`.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Branch: `main` and folder: `/(root)`.
5. Save and wait for the Pages URL to go live.
